Solución al Problema de Programación Dinámica : Bacterias

Enunciado

  Explicación

Este problema pide encontrar el camino menos costoso por el cual se puede recorrer e imprimir el valor de este costo. La restricción que define a este problema es el hecho de que solo se puede mover a la derecha o hacía abajo, lo cual hace que la programación sea más sencilla.

Al tomar en cuenta la programación dinámica y esta última restricción, se puede programar un arreglo en donde se va guardando el camino menos costoso conforme se va avanzando en las casillas del arreglo inicial. Esto se lleva acabo checando los valores que resultan de haberse movido hacia la derecha o hacia abajo en el paso anterior. (El máximo entre el valor de la casilla anterior arriba y anterior izquierda)



Flota sobre alguna celda del arreglo "caminoOptimo" después de crearse para ver el código que calcula su valor.

compuestoQuimico[Filas][Columnas]   caminoOptimo[Filas][Columnas]

 

  Código

int bacteriasRecorridoMinimo(int N, int M)

{

caminoOptimo[0][0] = compuestoQuimico[0][0];

for(int i = 1 ; i < M ; i++)

caminoOptimo[0][i] = compuestoQuimico[0][i] + caminoOptimo[0][i - 1];

for(int i = 1 ; i < N ; i++)

caminoOptimo[i][0] = compuestoQuimico[i][0] + caminoOptimo[i - 1][0];

for(int i = 1 ; i < M ; i++)

for(int j = 1 ; j < N ; j++)

caminoOptimo[i][j] = min(caminoOptimo[i][j - 1], caminoOptimo[i - 1][j]) + compuestoQuimico[i][j];

return caminoOptimo[N - 1][M - 1]

}
